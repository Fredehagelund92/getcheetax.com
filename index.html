CHEETAX

Data loader tool

Cheetax makes it easier to load data into EXASOL with a simple command-line interface.

Click here to learn more.

How it Works


1. Setup a cheetax

Once you install cheetax, run cheetax init, and start setting up data sources to your datawarehouse.

2. Configure data sources

Each analyst can use dbt to build their own isolated copy of the analytics schema. There's no need for your analysts to step on each others' toes any longer.

3. Load into your datawarehouse

Use digdag or cron to keep your production data warehouse running the latest and greatest version of your models.
